# ✧ GitScore: The Mystical GitHub Engineering Scryer ✧

GitScore is a comprehensive, mystical engineering evaluation platform that transmutes raw GitHub data into actionable developer insights. It analyzes your activities across the rift—commits, pull requests, stargazers, and constellations—to calculate a definitive "Mystical Score."

![GitScore Dashboard](https://git-score-v1.vercel.app/api/og?username=gaearon&score=94&grade=S)

## 🌌 Core Features

- **Mystical Scoring Engine**: Multi-dimensional analysis covering Technical Depth, Social Impact, Activity Consistency, and Innovation.
- **Dynamic Scying Dashboard**: Interactive visualizations of your language distribution, activity timelines, and star constellations.
- **Battle Arena**: Side-by-side profile comparison to see how explorers measure up in the coordinate clash.
- **Ethereal Badges**: Auto-generated SVG badges for your READMEs that reflect your current rank.
- **Watchlist Tracking**: Persistence scrying to monitor score improvements and profile evolution.
- **Spectral Exports**: Download your technical report as high-fidelity images or PDF documents.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App & Pages Router)
- **Styling**: Vanilla CSS with Tailwind-inspired motion utilities & [Framer Motion](https://www.framer.com/motion/)
- **Data Oracle**: GitHub GraphQL & REST APIs via [Octokit](https://github.com/octokit)
- **Persistence**: [Supabase](https://supabase.com/) (PostgreSQL & Edge Functions)
- **Analytics**: Google Analytics 4 + Custom Performance Monitoring
- **Infrastructure**: [Vercel](https://vercel.com/) (Edge Caching & Dynamic Hydration)

## 🚀 Setup Instructions

### Prerequisites
- Node.js 18.x or higher
- A GitHub Personal Access Token (PAT)
- A Supabase Project (URL & Anon Key)

### Local Dev Initiation
1. **Clone the coordinates**:
   ```bash
   git clone https://github.com/your-username/github-score.git
   cd github-score
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure the Ethereal Environment**:
   Create a `.env.local` file with:
   ```env
   GITHUB_TOKEN=your_pat_here
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
   NEXT_PUBLIC_GA_ID=your_ga4_id
   NEXT_PUBLIC_APP_URL=http://localhost:3000
   ```

4. **Summon the server**:
   ```bash
   npm run dev
   ```
